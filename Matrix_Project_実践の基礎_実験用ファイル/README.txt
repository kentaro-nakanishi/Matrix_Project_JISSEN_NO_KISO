これは

自動作曲システムMatrix Project

『Matrix Project　実践の基礎』に使う実験用プログラムです。


プログラムを動かすためには、以下の方法で環境構築をする必要があります。

プログラムが動作する環境構築の方法：

runtime.txtに書かれているバージョンのpythonをインストールしてください。

requirements.txtに書かれているライブラリをpipでインストールしてください。

また、以下のライブラリを自力でインストールしてください。

pretty_midi==0.2.9

（pipではインストールできません。）
